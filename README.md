## Tasks Completed

### Task 1: Dataset Visualization
- **Objective:** Explore and visualize the Iris dataset
- **Dataset:** Iris Dataset (built into seaborn)
- **Key Findings:** Setosa species is clearly separable from others using petal measurements; 150 flowers, 3 species, 50 each
- **Visualizations:** Scatter plot, histograms, box plots

### Task 3: Heart Disease Prediction
- **Objective:** Predict heart disease risk from patient health data
- **Dataset:** Heart Disease UCI Dataset (Kaggle)
- **Model:** Logistic Regression with StandardScaler
- **Accuracy:** 85.25% | **AUC:** 0.927
- **Key Findings:** Model effectively identifies at-risk patients with high separability

### Task 6: House Price Prediction
- **Objective:** Predict house prices using property features
- **Dataset:** House Prices - Advanced Regression Techniques (Kaggle)
- **Model:** Gradient Boosting Regressor
- **R² Score:** 0.89 | **MAE:** $19,128
- **Key Findings:** OverallQual (quality rating) and GrLivArea (living area) are top price predictors
